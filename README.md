## Найденные баги API (расхождения с документацией)

В ходе тестирования обнаружены следующие несоответствия документации:

| № | Ручка | Сценарий | По документации | Фактически |
|---|-------|----------|-----------------|------------|
| 1 | `DELETE /courier/:id` | Несуществующий id | 404 + сообщение без точки | 404 + сообщение с точкой |
| 2 | `DELETE /courier/` | Без ID | 400 | 404 |
| 3 | `POST /courier` | Дубликат логина | `Этот логин уже используется` | `... Попробуйте другой` |
| 4 | `POST /courier` | Только login+password (нет firstName) | 400 | 409 |
| 5 | `POST /courier/login` | Только login (без password) | 400 | 504 |
| 6 | `POST /courier/login` | Пустой запрос | 400 | 504 |
